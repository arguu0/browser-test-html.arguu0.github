Vivaldi Browser

Vivaldi is a highly customizable, privacy-focused web browser built for power users, developed by Vivaldi Technologies (founded by ex-Opera developers). Based on the Chromium engine, it features advanced tab management (stacking/tiling), built-in ad/tracker blockers, mail, calendar, and deep UI customization, available on desktop and mobile.
