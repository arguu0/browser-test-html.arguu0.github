Brave Browser

Brave is a free, open-source, privacy-focused browser based on Chromium (similar to Chrome) that automatically blocks ads, trackers, and malicious scripts by default, offering up to 3x faster browsing speeds. It features built-in privacy tools like Brave Shields, HTTPS Everywhere, and built-in Tor network support.
